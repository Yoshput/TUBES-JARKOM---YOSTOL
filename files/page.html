🔧 Arsitektur Sistem

Sistem ini mengimplementasikan arsitektur client-proxy-server dengan fitur caching dan quality of service.

Web Server
Port TCP 8000
Port UDP 9000
Multithreading
Proxy Server
Port TCP 8080
Caching Layer
Request Forwarding
Client TCP
HTTP GET Requests
Cache Aware
HTML Rendering
Client UDP
QoS Testing
Ping Statistics
Latency Measurement

📊 Fitur Utama

  • ✓ HTTP Parsing Manual (tanpa library eksternal)
  • ✓ Caching dengan thread-safe mechanism
  • ✓ Logging dengan timestamp dan IP client
  • ✓ Error handling (404, 500, 502, 504)
  • ✓ Multithreading untuk concurrent clients
  • ✓ UDP Echo untuk QoS monitoring
  • ✓ Graceful shutdown

🚀 Cara Menjalankan

Terminal 1: python webserver.py
Terminal 2: python proxy.py
Terminal 3: python client.py --mode tcp
Terminal 4: python client.py --mode udp

📈 Statistik Cache

Proxy server mencatat:

  • Total requests
  • Cache hits dan misses
  • Response time
  • Client IP addresses
